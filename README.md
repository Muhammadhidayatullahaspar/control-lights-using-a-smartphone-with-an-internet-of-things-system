# control-lights-using-a-smartphone-with-an-internet-of-things-system
This project uses esp32 as a controller that connects the lamp and smartphone with internet of things
Alat dan bahan yang digunakan :
1.	Laptop
2.	Modul esp32
3.	Relay
4.	Lampu
5.	Fitting lampu
6.	Kabel jumper
7.	Obeng
8.	Pisau
9.	Kabel usb
10.	Kabel ac
11.	Hp android
